Test Project

This is a simple test project to demonstrate how easy it is to set up a new project. In this exercise, we completed the following tasks:

  1. Set up the project folder and installed the initial packages
  2. Initialized Git
  3. Initialized Firebase
  4. Added some HTML code
  5. Linked the Bootstrap files
  6. Served and tested the project locally
  7. Packaged the project for deployment
  8. Deployed to Firebase

Later, I might decide to come back here and play around with this file some more, to test out some things. It's sort of like a code sandbox that I can use to practice Bootstrap components, styles, and more. If I want to change anything in this project, I just need to re-build and re-deploy my changes. It's that easy!
